
# Activités sur Paris : ParisParty

Application regroupant des activités sur Paris avec la possibilité de les ajouter en favoris et de les voir sur une carte.


## Lancer le projet

IOS :

```bash
  npx react-native run-ios
```
    
## Développé sur :

**IOS:** iPhone 13
